## ENV

| Variable         | Funktion                  |
| ---------------- | ------------------------- |
| DB_USER          | Datenbank Benutzer        |
| DB_SECRET        | Datenbank Passwort        |
| LDAP_USER        | AD (Admin-) Benutzer      |
| LDAP_PASSWORD    | AD Benutzer Passwort      |
| LDAP_URL         | AD IP                     |
| SESSION_NAME     | Session Cookie Name       |
| SESSION_PASSWORD | Session-internes Passwort |
